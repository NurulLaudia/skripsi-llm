# Sentiment Analysis using Fine-Tuned LLM (BERT)

## Deskripsi
Project skripsi ini bertujuan untuk menganalisis opini pelanggan terhadap produk retail online
menggunakan Large Language Model (BERT) yang telah di fine-tuning.

## Tujuan
- Mengklasifikasikan sentimen ulasan pelanggan (positif, negatif, netral)
- Menerapkan fine-tuning BERT untuk data ulasan produk skincare

## Teknologi
- Python
- HuggingFace Transformers
- BERT
- Pandas, NumPy, Scikit-learn

## Dataset
Dataset yang digunakan merupakan data ulasan produk retail online.
Repository ini hanya menyertakan data contoh (dummy).

## Hasil
Model menghasilkan akurasi dan performa yang baik dalam klasifikasi sentimen.
